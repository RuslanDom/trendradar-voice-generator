# 🎙️ TrendRadar Voice Generator
**100% Free Text-to-Speech app** — convert any text to natural speech using Microsoft Edge neural voices. Export as MP3. No ads, no paywalls, no limits.

✅ What's included:
• Windows (.exe) — ready to run, no installation needed
• Android (.apk) — side-load or install directly
• Linux — run with Python (pip install -r requirements.txt)

🎤 7 voices: Eric, Aria, Ryan, Guy, Jenny, Sonia, Natasha
🌍 American, British, Australian accents
🆓 100% free — no ads, no subscriptions, no limits

Built with Flet + Edge TTS + Python.
MIT License — use it however you want.
