# 🎙️ TrendRadar Voice Generator

**100% Free Text-to-Speech app** — convert any text to natural speech using Microsoft Edge neural voices. Export as MP3. No ads, no paywalls, no limits.

---

## ✨ Features

- 🎤 **7 high-quality voices** — Eric, Aria, Ryan, Guy, Jenny, Sonia, Natasha
- 🌍 **Multiple accents** — American, British, Australian
- 💾 **Export to MP3** — save anywhere on your device
- 🖥️ **Cross-platform** — Windows, Linux, Android
- 🆓 **100% Free** — no ads, no subscriptions, no hidden costs
- 🔓 **Open Source** — MIT license
